# BedrockRooms YMax Addon

Authors: Tuma, Tymamatyty

## EN
Addon that checks block distribution percentages on Y-max level.

- Computes Y-max distribution statistics.
- Mostly useful on HolyWorld-like server environments.

## RU
Аддон для проверки процентного распределения блоков на уровне Y-max.

- Считает статистику распределения на Y-max.
- Практически полезен в основном на серверах типа HolyWorld.
